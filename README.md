# XRD analysis marimo app

Этот репозиторий содержит интерактивное приложение для обработки дифрактограмм. Основной файл приложения - `vkrr.pyd.py`; это marimo-ноутбук, поэтому его нужно запускать через marimo, а не как обычный Python-скрипт.

## Что нужно установить

Перед запуском лучше создать отдельное Python-окружение и установить библиотеки, которые используются в приложении:

```bash
pip install marimo polars lets-plot numpy scipy
```

После установки приложение можно открыть командой:

```bash
marimo edit vkrr.pyd.py
```

Если нужен запуск как приложения:

```bash
marimo run vkrr.pyd.py
```

## Входные данные

Файл `HOMBIFINE.txt` - пример входных данных для анализа. Для него уже подобраны наиболее удачные значения параметров, поэтому текущие дефолтные настройки приложения ориентированы именно на этот набор данных.

Формат входного файла простой: две числовые колонки, где первая колонка - координата по оси `2θ`, а вторая - интенсивность.
